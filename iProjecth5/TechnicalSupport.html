iProject version:1.0

iProject is an app for developer private project management. In this app you can manage your project's collection progress, project scheduling, project collection management and more.

1. Project management

You can create projects, add project payment information, and view historical payment lists so you can clearly see the progress of your project's payment;


2. Project plan

You can create a work plan for the corresponding project and share it with the customer;


3. Backup and clearing of project data;
