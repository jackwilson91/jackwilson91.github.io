PartTimeJobs version:1.0

This is an app for developers to record part-time information, you can use this app to save your part-time project information, project progress information and project payment information.

1. Project information:

You can save the project's name, type, project content, contacts, contact phone number and other information, you can click on the project list and then add or view the project progress, project payment information;

2. Project progress:

You can select the specific project and fill in the progress information of the project: the completed project content, the current project progress; you can also view the specific project progress list;

3. Project income:

You can view the item's payment information after selecting the specific item: payment amount, payment time, payment note;

4. More:

You can view the report of the project payment according to the project type. The report is displayed in a pie chart, which is simple and intuitive.

You can also export the payment report for the project in the format xls, which includes the project name, project type, payment amount, payment time, and payment notes.
